# Café Explorer

Café Explorer is a lightweight Flask web application that displays curated cafés with details such as opening hours, coffee quality, wifi strength, and power‑outlet availability. Each café is loaded from a CSV file and presented through a modern, responsive interface using Bootstrap.

## Features

- CSV‑based café data loading  
- Individual café cards with ratings and quick‑access map links  
- Random café‑themed images fetched via API  
- Clean, English‑only UI with light‑mode design  
- Simple form for adding new cafés to the dataset  

## Tech Stack

- Flask  
- Bootstrap 5  
- Jinja2 templates  
- External image API for café previews  
- CSV for data storage
